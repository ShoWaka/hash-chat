# Features

LINE 風の UI を用いて、入力テキストをハッシュ化したものを表示する

ハッシュ化アルゴリズムを

- SHA-1
- SHA-256
- SHA-384
- SHA-512
  から選択してハッシュ化できる。

# Requirement

"hoge"を動かすのに必要なライブラリなどを列挙する

- material-tailwind/react 2.0.6
- react-chat-elements 12.0.10

# Installation

```bash
pip install @material-tailwind/react
pip install react-chat-elements 12.0.10
```
